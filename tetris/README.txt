"tetris.exe" can only be played if the dll files
and the src folder are in the same directory

how to play:
- CONTROL SCHEME 1:
	left arrow: move left
	right arrow: move right
	down arrow: move down
	up arrow: hard drop
	space: rotate
- CONTROL SCHEME 2:
	left arrow: move left
	right arrow: move right
	down arrow: move down
	up arrow: rotate
	space: hard drop
